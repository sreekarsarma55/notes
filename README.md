# 📚 Notes

A personal knowledge base for study notes, revision material, cheat sheets, and worked problems across different subjects.

## 🗂️ How this repo is organised

**One branch per subject.** Each subject lives on its own branch so that material stays isolated, easy to browse, and easy to grow independently over time. The `main` branch holds only this guide and the subject index.

| Subject | Branch | Status |
|---------|--------|--------|
| Machine Learning Techniques (MLT) | [`mlt`](https://github.com/sreekarsarma55/notes/tree/mlt) | Weeks 1–4 (Quiz 1 scope) |

> To add a new subject later, create a new branch (e.g. `dsa`, `stats`, `dbms`) and follow the same folder layout shown below.

## 📁 Layout within a subject branch

```
<subject>/
├── README.md          # subject index + how to use these notes
├── NN-topic.md        # one file per topic/week, fully explained
├── cheat-sheet.md     # condensed one-page formula sheet for last-minute revision
├── mock-solutions.md  # worked solutions to mock/practice papers
└── images/            # diagrams and figures referenced by the notes
```

## ✍️ Conventions

- **Markdown** for all notes, with plain-text math blocks (renders everywhere, including GitHub and mobile).
- **Images only where they add understanding** — diagrams for intuition, not decoration.
- Each topic file follows the same shape: *intuition → formulas → worked example → common exam traps*.
